# Task 2 - Home Page & Product Listing

## Project Overview
This is the Week 2 submission for the Frontend Development Internship. The task implements the Home Page and Product Listing Page for an e-commerce marketplace.

## Technologies Used
- HTML5 (Semantic Elements)
- CSS3 (Flexbox, Grid)
- Google Fonts (Inter)
- Font Awesome Icons

## Features Implemented

### 1. Hero Section
- Eye-catching banner with gradient background
- "Hot Deals" tag
- Call-to-action "Shop Now" button with hover animation
- Promotional image with shadow effect

### 2. Featured Categories
- 4 category cards: Electronics, Fashion, Home & Living, Beauty
- Hover effects with lift animation
- Category images with consistent sizing
- "View All" link to browse all categories

### 3. Product Listing
- Grid layout with 6 product cards
- Each product includes:
  - Product image
  - Product name
  - Price with styling
  - "Buy Now" button with hover effects
- Product badges: NEW, SALE, HOT
- Hover effects on product cards (lift + shadow)

### 4. Pagination
- Page number navigation (1, 2, 3, 4)
- "Next" button for more products
- Active page highlighting

## Technical Specifications
- ✅ Semantic HTML5 elements
- ✅ External CSS only (no inline styles)
- ✅ CSS Grid for product and category layouts
- ✅ Flexbox for header and alignment
- ✅ BEM naming convention
- ✅ Pixel-perfect desktop implementation
- ✅ Responsive design (desktop, tablet, mobile)
- ✅ Hover effects on interactive elements
- ✅ Sticky header
- ✅ Footer fixed at bottom
