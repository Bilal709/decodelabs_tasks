# E-Commerce Frontend Design - Task 1

## Project Overview
This is the Week 1 submission for the Frontend Development Internship task. The project implements a pixel-perfect static webpage design for an e-commerce marketplace inspired by Alibaba-style layout.

## Technologies Used
- HTML5 (Semantic Elements)
- CSS3 (Flexbox, Grid, BEM Methodology)
- Google Fonts (Inter)
- Font Awesome Icons

## Features Implemented (Week 1)

### Header
- Brand logo with tagline
- Search bar with search icon
- Navigation menu with icons:
  - Home (Active)
  - Shop
  - Deals
  - Account
  - Cart (with badge counter)

### Footer
- 5-column layout:
  - About Us
  - Customer Support
  - Policies
  - Connect (Social Media)
  - Download App (App Store & Play Store)
- Payment method icons
- Copyright information

## Technical Specifications
- ✅ Semantic HTML5 elements
- ✅ External CSS only (no inline styles)
- ✅ CSS Grid for macro layout
- ✅ Flexbox for micro component alignment
- ✅ BEM naming convention
- ✅ Sticky header
- ✅ Footer fixed at bottom
- ✅ Responsive design for desktop, tablet, and mobile
